This dashboard allows you to manage user accounts and user profiles in the fundraising system. You can create, view, update, suspend, and search both accounts and profiles from the navigation menu.

User Account Functions

  • Create user account to store new personal information
  • View user account details
  • Update user account information
  • Suspend user account to stop access
  • Search user account by personal information

User Profile Functions

  • Create user profile to define a new role
  • View user profile and permissions
  • Update role settings
  • Suspend a specific user profile
  • Search user profile by role
